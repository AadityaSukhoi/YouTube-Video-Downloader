# YouTube-Video-Downloader
A sleek and simple **Streamlit web app** that lets you download YouTube videos in just a few clicks. No ads. No nonsense. Just pure downloading power.

---

## 🚀 Live Demo

👉 [Click here to try it out!](https://yt-vid.streamlit.app)  

---

## 📌 Features

- 🔗 Paste a YouTube video link
- 📥 Download video in the best available quality
- 🧠 Auto-detects video metadata (title, thumbnail)
- ⚡ Powered by `yt-dlp` and `ffmpeg`
- 🕶️ Minimal and clean UI

---

## 🛠 How It Works

This app uses:
- `yt-dlp` to fetch and stream video content
- `ffmpeg` for any video/audio post-processing (installed automatically via `packages.txt`)
- `Streamlit` to build and deploy the web interface

---

## 🧰 Requirements

If you're running locally:

```bash
pip install -r requirements.txt
```

---

## How to Run Locally?

```bash
git clone https://github.com/AadityaSukhoi/Youtube-Video-Downloader.git
cd Youtube-Video-Downloader
streamlit run app.py
```
